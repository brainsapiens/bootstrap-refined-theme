Spacing

First of all, for through spacing consistency now $spacer is used as a bottom indent on all those components that it did not apply to before, namely:

  • <p> paragraphs;
  • .form-group element;
  • Alerts component;
  • Breadcrumb component;
  • Changed all HTML headings, from <h1> through <h6> bottom indent from ($spacer / 2) to $spacer.

The 4 size multiplier changed to 2 to maintain an equal rhythm:

.p-1 { padding: ($spacer * .25) !important; }
.p-2 { padding: ($spacer * .5) !important; }

.p-3 { padding: ($spacer * 1) !important; }
.p-4 { padding: ($spacer * 1.5 2 ) !important; }
.p-5 { padding: ($spacer * 3) !important; }

The idea is to use small sizes 1 and 2 mainly for the local elements (left/right spacing before/after icons, for example), and larger sizes 3, 4 and 5 for the global blocks (top/bottom spacing before/after sections, for example). This paragraph is for guidance only.
